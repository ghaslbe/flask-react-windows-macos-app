# ⚡ Schnellstart - Benutzerverwaltung bauen

## 🪟 Windows (Ein-Klick)
```cmd
build_all.bat
```
**Fertig!** EXE ist in `dist/Benutzerverwaltung.exe`

---

## 🍎 macOS (Zwei Kommandos)
```bash
python build_macos.py
python create_dmg.py
```
**Fertig!** DMG ist `Benutzerverwaltung.dmg`

---

## 🎯 Das war's!
- **Windows:** `dist/Benutzerverwaltung.exe` verteilen
- **macOS:** `Benutzerverwaltung.dmg` verteilen
- Empfänger: Doppelklick → Browser öffnet automatisch
- Daten landen in `Documents/Benutzerverwaltung/`
